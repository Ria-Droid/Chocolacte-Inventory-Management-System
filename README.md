# Gloabal Chocolate  Inventory Management System

## Overview

A C++ inventory management system developed using object-oriented programming principles. The application manages chocolate products, inventory, customer orders, discounts, and tax calculations.

## Features

- Inventory management
- Customer order processing
- Discount management
- Tax calculations
- Object-oriented design

## Technologies

- C++
- Code::Blocks
- Object-Oriented Programming

## How to Run

1. Open `CodeCoven.cbp` in Code::Blocks.
2. Build the project.
3. Run the application.
